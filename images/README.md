# Images Folder

Add your photos here! Recommended:

## Photo Types Needed

1. **Profile Photo** - Professional headshot of Iesha
2. **Work Photos** - Before/after shots of hair styles
3. **Gallery Images** - Various completed styles:
   - Braids
   - Locs
   - Color work
   - Roller sets
   - Ponytails
   - Quick weaves

## Recommended Specs

- **Format**: JPG or WebP (smaller file size)
- **Size**: 1200px wide minimum for quality
- **Naming**: Use descriptive names like `braids-1.jpg`, `locs-retwist.jpg`

## Tips

- Good lighting makes all the difference
- Consistent backgrounds look more professional
- Before/after shots are great for portfolios
- Get client permission before posting their photos
